My Work

Software Development - DCM Project

  • As a member of the team working on the PACEMAKER project, my role was to contribute to the development of the Device Controller-Monitor (DCM).
  • I was responsible for developing the user interface for the DCM, which included a welcome screen, the ability to register and login users, and essential aspects of the interface in accordance with the PACEMAKER requirements.
  • Developed interfaces for presenting all of the pacing modes to the user and made provision for storing programmable parameter data.
  • My programming language of choice for this project was Python, which I used to create a simple GUI and to establish stable serial communication between the DCM and the pacemaker device

Digital Systems: Hardware Implementation of an Image Decompressor

Created an image decompression solution using the Altera DE2 board and hardware description language. Established Finite State Machine (FSM) systems for the decompression of a 320x240 image. Successfully optimized the use of onboard multipliers, achieving a 78% utilization rate, for interpolation and color space conversion. Wrote and debugged over 1000 lines of Verilog code using ModelSim extensively for testing and verifying the functionality of the code.
